Example:
MaxElements (A, n)
- Input:
	- array of integers A
	- size of array n
- Output:
	- Maximum element in A

```
	CurrentMax  A[0]
	for(i = 1; i < n-1; i++)
	 do:
	   if(A[i] > CurrentMax) then:
	     CurrentMax  A[i]
       return CurrentMax
```
